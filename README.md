# react-native-trusted-device-v2

Trusted device for react native

## Installation

```sh
npm install react-native-trusted-device-v2
```

## Usage

```js
import Fazpass from 'react-native-trusted-device-v2';

// ...

const meta = await Fazpass.instance.generateMeta();
```
